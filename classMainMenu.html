Detailed Description

Handles the display and interaction logic for the game's main menu.

Manages menu navigation, rule screens, and delegates game state changes.

bool MainMenu::showMainMenu ( GameState &  state,

Manages the main menu interaction loop.

Displays the main menu and handles player choices.

Parameters
stateReference to the game's state (modified on START).
runningFlag controlling the game loop (set to false on QUIT).
Returns
bool Whether to continue running the game (false if QUIT is selected).

The loop continues until "Quit" or "Start" is selected.

  • Calls selectDeck() if "Start" is chosen.
  • Shows the rules screen if "Rules" is selected.
  • Stops the game if "Quit" is selected.
